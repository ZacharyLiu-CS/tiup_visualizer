## 详情模态窗口

### 基本信息区
显示集群的元数据:
- Cluster Type (tidb/tikv)
- Version (v8.1.0)
- Deploy User (root)
- Dashboard URL (可点击)
- Grafana URL (可点击)

### 组件列表
表格显示所有组件:
- **ID**: 组件标识 (IP:端口)
- **Role**: 组件角色,带颜色徽章
  - tikv (蓝色)
  - pd (粉色)
  - tidb (绿色)
  - prometheus (橙色)
  - grafana (紫色)
  - alertmanager (红色)
- **Host**: 主机 IP
- **Ports**: 端口号
- **Status**: 状态,带颜色徽章
  - Up (绿色)
  - Down (红色)
  - N/A (灰色)
- **Data Dir**: 数据目录
- **Deploy Dir**: 部署目录

---

## 颜色说明

| 颜色 | 含义 |
|------|------|
| 🔵 蓝色边框 | 选中的主机 |
| 🔷 紫色边框 | 选中的集群 |
| 🟢 绿色边框 | 高亮的项 (关联关系) |
| 🔵 蓝色虚线 | 主机到集群的连接 |
| 🔷 紫色虚线 | 集群到主机的连接 |

---

## 状态徽章

### 组件状态
- `Up` - 正常运行 (绿色)
- `Down` - 已停止 (红色)
- `N/A` - 状态未知 (灰色)
- `Up|L|UI` - 运行中,Leader,有UI (绿色)

### 组件角色
- `tikv` - TiKV 存储节点 (蓝色)
- `pd` - Placement Driver (粉色)
- `tidb` - TiDB 服务器 (绿色)
- `prometheus` - 监控系统 (橙色)
- `grafana` - 可视化面板 (紫色)
- `alertmanager` - 告警管理 (红色)

## 键盘快捷键

目前不支持键盘快捷键,所有操作通过鼠标完成。

未来可能添加:
- `Esc` - 关闭详情窗口/清除选择
- `1-9` - 快速选择主机/集群
- `Space` - 刷新数据

## 提示和技巧

1. **清除选择**: 点击 "Clear Selection" 按钮或再次点击已选中的项

2. **查看完整路径**: 鼠标悬停在数据目录或部署目录上可能显示完整路径(如果浏览器支持)

3. **打开外部链接**: Dashboard 和 Grafana URL 可以直接点击在新标签页打开

4. **刷新数据**: 页面会自动从 TiUP 获取最新数据,如需手动刷新可刷新浏览器页面

5. **多集群环境**: 如果有很多集群,可以使用浏览器的查找功能 (Ctrl+F) 快速定位

## 常见问题

**Q: 为什么看不到连接线?**
A: 确保选中了主机或集群,连接线只在有选择时显示。

**Q: 如何同时查看多个集群?**
A: 目前只支持单选,一次只能选择一个主机或集群。

**Q: 组件状态多久更新一次?**
A: 每次页面加载或刷新时从 TiUP 获取最新状态。

**Q: 能否编辑集群配置?**
A: 当前版本只支持查看,不支持编辑。需要编辑请使用 TiUP 命令行工具。
